## Description (max 4000 chars)
Verbinal is a native macOS desktop companion for the CANFAR Science Portal — Canada's platform for astronomical research computing.

Manage your interactive science sessions directly from your Mac without a web browser. Launch notebooks, desktops, and visualization tools, monitor platform resources, and track your storage usage — all from a clean, native interface.

FEATURES

- Launch and manage interactive sessions (Notebook, Desktop, CARTA, Contributed, Firefly)
- Monitor session status with auto-refresh polling
- View platform CPU and RAM availability in real time
- Track VOSpace storage quota and usage
- Quick re-launch from recent session history
- View session events and container logs
- Secure login with credentials stored in macOS Keychain

BUILT FOR SCIENTISTS

Verbinal is designed for researchers using the Canadian Advanced Network for Astronomical Research (CANFAR) and the Canadian Astronomy Data Centre (CADC). It provides the same session management capabilities as the CANFAR web portal in a fast, native macOS app.

PRIVACY

No data collection, no analytics, no telemetry. Your credentials stay in your Keychain. All communication goes directly to CANFAR services over HTTPS.

OPEN SOURCE

Verbinal is open source under the Mozilla Public License 2.0. View the source code, report issues, or contribute at github.com/szautkin/canfar-macos.
